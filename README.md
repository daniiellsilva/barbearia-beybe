<h1>💈 Barbearia Beybe do Cortte</h1>
<div>
    <p>
  Bem-vindo ao projeto Barbearia Beybe do Cortte ✂️🔥
</p>
<p>
  Um site desenvolvido para facilitar o contato entre clientes e a barbearia, reunindo tudo em um só lugar: redes sociais, localização, Wi-Fi e uma galeria mostrando o trabalho do barbeiro.
</p>
</div>


<div>
  <strong>✨ Sobre o Projeto</strong>

  O projeto foi criado pensando na praticidade e experiência do cliente.
  <br>
  <br>
  <strong>Dentro do site é possível:</strong>
</div>
<ul>
  <li>📲 Acessar o WhatsApp da barbearia</li>
  <li>📸 Ir direto para o Instagram</li>
  <li>📍 Visualizar a localização da barbearia</li>
  <li>📶 Consultar o nome e senha do Wi-Fi</li>
  <li>🖼️ Ver uma galeria com alguns trabalhos realizados</li>
</ul>

Toda a interface foi personalizada de acordo com o estilo desejado pelo barbeiro, buscando transmitir identidade visual, modernidade e presença digital.
